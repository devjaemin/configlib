2023-02-08

1. IDAttribute에 Default 방식 추가
    - 이제 속성에 ID 특성을 지정할 때 매개변수 없는 생성자를 사용하여 지정할 경우 해당 속성의 이름이 ID로 사용됨. (대소문자 구분)

2. Null 검사 방식 변경
    - null 값이 저장될 때 문자열 "Null"로 저장되는 방식에서 "\0"이 저장되는 방식으로 변경됨.
    이에 따라 속성 값을 불러올 때 TryParse() 메서드 다음으로 값이 "\0"이고, 속성의 타입이 참조 형식이거나 Nullable 형식인 경우 null로 설정됨.

3. Config 파일의 상단에 주석 추가

4. Config 파일의 각 라인을 읽어올 때 라인의 마지막 위치에 콜론이 있는 경우에 대한 검사 삭제 및 configurations로 옮길 때도 Substring에서 잘라낼 길이는 지정하지 않도록 변경.
